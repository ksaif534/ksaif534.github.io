Introduction

Thank you for purchasing the Elevate Template Package! This comprehensive installation guide will help you get started with setting up and using any of the templates included in this package.

Requirements

To use any of the Elevate templates, you'll need:

  • A code editor (VS Code, Sublime Text, Atom, etc.)
  • Basic knowledge of HTML, CSS, and JavaScript
  • Modern web browser (Chrome, Firefox, Safari, Edge)
  • Local development server (optional but recommended)

Note: All templates are built with modern web standards and are fully responsive, ensuring they work well on all devices.

Quick Installation

Follow these simple steps to quickly get started with any of the Elevate templates:

  1. Download and Extract: Unzip the downloaded package to your desired location
  2. Choose a Template: Navigate to the template folder you want to use (elevate-alt, elevate-colorful, or elevate-portfolio)
  3. Open in Browser: Open the index.html file in your web browser to view the template
  4. Start Customizing: Edit the HTML, CSS, and JavaScript files to customize the template

Tip: For the best development experience, we recommend setting up a local development server as described in the Local Development section.

Decide which template best suits your needs:

Local Development

For the best development experience, we recommend using a local development server. Here are a few options:

Using Visual Studio Code with Live Server Extension

  1. Install Visual Studio Code
  2. Install the "Live Server" extension
  3. Right-click on index.html and select "Open with Live Server"

Using Python's Built-in HTTP Server

  1. Open a terminal/command prompt
  2. Navigate to the template directory
  3. Run one of the following commands:
    • Python 3: python -m http.server
    • Python 2: python -m SimpleHTTPServer
  4. Open your browser and go to http://localhost:8000

Using Node.js and http-server

  1. Install Node.js
  2. Install http-server globally: npm install -g http-server
  3. Navigate to the template directory in terminal/command prompt
  4. Run: http-server
  5. Open your browser and go to the URL provided in the terminal

Troubleshooting

Common Issues and Solutions

Images Not Loading

  • Check that the image paths in the HTML files are correct
  • Ensure all image files are in the correct directories
  • Verify that image file names match exactly (case-sensitive)

JavaScript Features Not Working

  • Check the browser console for any errors
  • Ensure all JavaScript files are properly linked in the HTML
  • Verify that you're using a modern browser that supports ES6 features

Responsive Design Issues

  • Make sure the viewport meta tag is included in the HTML head
  • Check that you're using the latest version of your browser
  • Verify that responsive.css is properly linked in the HTML

Template-Specific Issues

Elevate-Alt: Sidebar Toggle Not Working
  • Check that sidebar.js is properly included in the HTML
  • Verify that the sidebar elements have the correct IDs and classes
  • Check the browser console for any JavaScript errors
Elevate-Portfolio: Portfolio Filtering Not Working
  • Check that portfolio.js is properly included in the HTML
  • Verify that portfolio items have the correct data-category attributes
  • Check the browser console for any JavaScript errors

Important: If you encounter any issues not covered here, please contact our support team at support@elevate-templates.com.
