Preview Images

Preview images are crucial for showcasing your template package on Envato/ThemeForest. They are the first thing potential buyers see, so they should accurately represent your templates and highlight their key features.

Main Preview Image

  • Dimensions: 590px × 300px
  • Format: JPG or PNG
  • Purpose: This is the main image displayed on your item page

Important: The main preview image must accurately represent the template package. It should showcase the best features and design elements from all three templates (Elevate-Alt, Elevate-Colorful, and Elevate-Portfolio).

Main Preview Image Guidelines

  • Must be high quality and visually appealing
  • Should represent the actual templates, not mockups
  • No text overlays allowed
  • Should be properly cropped and aligned
  • Must not include any watermarks or logos

Additional Preview Images

  • Dimensions: 590px wide (height can vary)
  • Format: JPG or PNG
  • Purpose: To showcase different aspects and features of your templates

Thumbnail Image

Thumbnail Requirements

  • Dimensions: 80px × 80px
  • Format: JPG or PNG
  • Purpose: Small icon displayed in search results and category pages

Thumbnail Guidelines

  • Should be simple and recognizable
  • Consistent with your brand
  • No text (too small to be readable)

Sample SEO Information Section

## SEO Information

### SEO Features Implemented
The Elevate Template Package includes the following SEO features:

- Semantic HTML5 structure
- Proper heading hierarchy (h1, h2, h3, etc.)
- Meta tags for title, description, and keywords
- Open Graph meta tags for social sharing
- Schema.org markup for improved search engine understanding
- Mobile-friendly responsive design (improves SEO ranking)
- Fast loading times (optimized code and assets)

### Customizing SEO Elements
To customize the SEO elements in the templates:

1. **Page Title and Meta Description**
   - Open `index.html`
   - Find the `` tag and update it with your page title
   - Find the `<meta name="description">` tag and update it with your page description

2. **Open Graph Meta Tags**
   - Open `index.html`
   - Find the Open Graph meta tags (starting with `<meta property="og:">`)
   - Update them with your information for social sharing

3. **Schema.org Markup**
   - Open `index.html`
   - Find the Schema.org markup (in the `<script type="application/ld+json">` section)
   - Update it with your information for improved search engine understanding

### SEO Best Practices
When using these templates, follow these SEO best practices:

- Use descriptive, keyword-rich page titles
- Write compelling meta descriptions
- Use descriptive alt text for images
- Create descriptive URLs
- Optimize content with relevant keywords
- Ensure fast loading times
- Make sure your website is mobile-friendly</code></pre>
